# 🍽️ Zomato Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Zomato restaurant dataset to uncover meaningful insights about restaurants, ratings, cuisines, online ordering, table booking, and pricing trends.

The analysis is carried out using Python in a Jupyter Notebook with various data preprocessing, cleaning, and visualization techniques.

---

## 🎯 Objectives

- Clean and preprocess the Zomato dataset.
- Handle missing values and duplicate records.
- Analyze restaurant ratings and votes.
- Explore popular cuisines and restaurant types.
- Study the impact of online ordering and table booking.
- Analyze pricing trends across restaurants.
- Generate visualizations to derive business insights.

---

## 📂 Dataset

The dataset contains information such as:

- Restaurant Name
- Online Order Availability
- Table Booking Availability
- Rating
- Votes
- Restaurant Type
- Cuisines
- Approximate Cost for Two People
- Reviews
- Restaurant Location
- City
